Procedimentos realizados

O primeiro passo do procedimento deste primeiro experimento consistiu nas configurações da conta login em um computador da UFABC, considerando o Sistema Operacional Ubuntu. Para isso, foi necessário o login inicial na máquina considerando um usuário e senha genéricos da UFABC. Na interface inicial, considerou-se os seguintes passos: entrar em "System Settings" (ou Configurações) e, logo após, clicar em "User Accounts" (ou Usuários). Na interface que se abre, deve-se clicar no botão "Unlock" (ou Desbloquear) no canto superior direito, o que leva à possibilidade de criação de uma nova conta, clicando no botão inferior com um símbolo "+", após a realização da devida autenticação do login atual. Feito isso, foi aberta uma nova janela para adicionar uma nova conta ao computador, onde foi considerado o login de Admnistrador em "Account Type" (ou Tipo da Conta) e foi digitado o sobrenome do aluno no campo "Full Name" (ou Nome Completo), sendo que o campo inferior já era preenchido automaticamente copiando o campo anterior, só que em caracteres minúsculos. Por fim, foi clicado em "Add" (ou Adicionar) e a senha pôde ser definida (a mesma senha foi considerada para todos os alunos da turma, considerando a sugestão do professor). Realizando o logout da conta genérica, foi feito o login na conta recém criada por cada um dos alunos.

Após a criação da nova conta e do devido login utilizando a mesma, foi aberto o terminal de comandos na busca do menu do sistema Ubuntu, onde foram realizadas instalações e atualizações com os comandos explicitados no passo a passo do laboratório em questão. Os comandos passados ao terminal foram os seguintes:

sudo apt-get update
sudo apt-get install build-essential
sudo apt upgrade
sudo apt-get install g++ freeglut3-dev buildessential libx11-dev libxmu-dev libxi-dev libglu1-mesa libglu1-mesa-dev
sudo apt-get install geany

Continuando com o passo a passo, na Área de Trabalho de cada um foi criada uma pasta com o primeiro nome do aluno, com uma sub-pasta denominada "lab01", onde foi copiado o arquivo lesson5.cpp, disponibilizado na página da disciplina no Moodle. Com o terminal aberto considerando esse diretório o arquivo cpp pôde ser compilado com o seguinte comando: gcc -o test lesson5.cpp -lglut -lGL -lGLU. Para testar o funcionamento do arquivo, considerou-se o comando ./test. O programa apresentou uma imagem, que foi capturada.

Imagens do funcionamento do programa foram capturadas, bem como imagens da modificação de cabeçalho da exibição, considerando nome do aluno e data, conforme orientações passadas pelo professor em aula.

Análises e conclusões

Assim, no final dos procedimentos, notou-se que a instalação das bibliotecas, como também a execução do programa OpenGL pode ser feito de forma relativamente simples, através do terminal Linux. Observou-se que o programa consistia de uma pirâmide e de um cubo, que rotacionam várias vezes por segundo. Analisando o programa em C++, verifica-se que estas rotações não possuem um limitador programado, o que significa que quanto mais potente for o computador, mais rotações serão observadas.

Outro ponto notável é que, ao se modificar o cabeçalho da exibição, se percebeu que a codificação do programa OpenGL não é tão simples, tendo que se observar com cuidado os comandos para modificá-lo da forma correta.
