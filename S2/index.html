Sistema Massa Mola Amortecedor

Para o projeto, optou-se por representar de forma didática um fundamento teórico estudado em disciplinas passadas, o sistema massa-mola-amortecedor, visto na disciplina de Sistemas de controle 1, obrigatória para a Engenharia da Informação, Engenharia de Instrumentação, Automação e Robótica e também para a Engenharia Aeroespacial, além de sem um dos exemplos mais clássicos de representação de um sistema de controle.

As equações que modelam o sistema podem ser conferidas na Modelagem funcional

Para compilar o programa, o usuário deve ter em mãos os dois de código fonte usados, mma.c e gui.c. Para compilar o código mma.c, deve usar o comando gcc -o mma mma.cpp -lglut -lGL -lGLU -lm -SOIL e executar usando o comando ./mma. Para compilar o código gui.c, deve-se usar o comando gcc -Wno-format -o gui gui.c -Wno-deprecated-declarations -Wno-format-security -lm `pkg-config --cflags --libs gtk+-3.0` -export-dynamic e executar usando o comando ./gui. Alternativamente, pode-se executar usando o comando ./gui & ./mma && fg, para rodar os dois programas no mesmo terminal. Deve-se garantir também que tenha instalado as bibliotecas g++, freeglut3-dev, buildessential, libx11-dev, libxmu-dev, libxi-dev, libglu1-mesa, libglu1-mesa-dev e libsoil-dev.
