Neste primeiro programa apenas se realizou a execução dele utilizando do comando gcc -o checker checker.c -lglut -lGL -lGLU && ./checker e se visualizou o resultado, que é mostrado na imagem a seguir, como também se deu uma olhada no código do programa.

Texbind.c

Neste programa ao execeutá-lo vemos que há a utilização de duas testuras, basicemente as diferenças percebidas entre elas é que em um temos quadrados menores brancos enquanto na outra há quadrados maiores vermelhos. Assim nota-se que ao se utilizar duas texturas tem-se duas representações diferentes na imagem. Para que se possa aplicar ambas as texturas em um mesmo objeto devemos escolher áreas diferentes do objeto para cada textura.

TextureRGB.c

Neste programa o resultado obtido é uma pirâmide com uma textura de uma imagem de olho no qual pode mudar as cores entre azul, verde e vermelho. Isso pode ser modificado utilizando um menu que é apresentado quando clicamos com o botão direito do mouse.
